# Local URL Shortener

A lightweight Python application that generates short aliases for URLs and stores them locally.

## Overview

Local URL Shortener is a simple console application that creates unique short links without using an online service.

The project demonstrates modular programming, dictionaries, hashing, validation, and local data management.

## Features

- Generate short links
- Store links locally
- Validate URLs
- Resolve aliases
- Display stored links
